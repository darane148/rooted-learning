# Week 1 - Day 1 - Linux Understanding

## 🧠 What is Linux?

Linux is an open-source operating system that manages hardware resources and enables users to interact with the system through a command-line interface (CLI) or graphical interface (GUI).

It is widely used in servers, cybersecurity, and development environments due to its flexibility, security, and powerful control over system operations.

---

## ⚙️ How Linux Works?

Linux follows a layered architecture:

1. **Kernel**
   - The core component of the operating system  
   - Manages CPU, memory, and hardware devices  

2. **Shell**
   - Acts as an interface between the user and the kernel  
   - Accepts user commands and executes them  

3. **File System**
   - Organizes files and directories in a structured hierarchy  
   - In Linux, everything is treated as a file  

4. **User Space**
   - Where applications and users interact with the system  

---

## 📂 Important Linux Concepts

- File System Structure (/home, /etc, /var)  
- File Permissions (read, write, execute)  
- Users and Groups  
- Processes and Services  
- Package Management  
- Basic Networking in Linux  

---

## 🔐 Why Linux is Important for Cybersecurity?

- Most servers and systems run on Linux  
- Many security tools are built specifically for Linux (e.g., Kali Linux)  
- Provides low-level system access and control  
- Widely used in penetration testing and ethical hacking  
- Helps in understanding system vulnerabilities and exploitation techniques  

---

## 🎯 What to Learn for Cybersecurity?

To effectively use Linux in cybersecurity, focus on:

- Basic commands (pwd, ls, cd, mkdir)  
- File permissions (chmod, chown)  
- Process management (ps, top, kill)  
- Networking commands (ifconfig, netstat, ss)  
- Log analysis (/var/log)  
- Shell scripting basics
