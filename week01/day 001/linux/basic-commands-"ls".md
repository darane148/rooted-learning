# Week 1 - Day 1 - Linux Basic Command: ls

## 🧠 What is `ls`?

`ls` (list) command is used to display files and directories in the current location.

It is one of the most basic and frequently used commands in Linux.

---

## 💻 Commands Used

```bash
ls
```
```bash
ls -a
```
```bash
ls -l
```
```bash
ls -lrt

```
---
## 🔍 Explanation

-🔹 `ls`
Displays visible files and directories

-🔹 `ls -a`
Shows all files including hidden files
Hidden files start with . (e.g., .bashrc, .config)

-🔹 `ls -l`
Displays detailed information:
File permissions
Owner
Size
Date & time

-🔹 `ls -lrt`
Lists files in long format
Sorted by time (oldest first)
