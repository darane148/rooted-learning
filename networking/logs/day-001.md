## Day 001 – 2026-04-04

**Focus:** Nmap scanning techniques
**Environment:** Kali Linux 2024.1 → Metasploitable2 (local network, 192.168.56.101)

---

### Commands Run

```bash
# Confirm target is alive (ICMP ping)
ping -c 3 192.168.56.101

# Basic TCP connect scan (noisy, full handshake)
nmap -sT 192.168.56.101

# SYN scan (stealthier — half-open, requires root)
sudo nmap -sS 192.168.56.101

# Service version detection
sudo nmap -sV 192.168.56.101

# OS detection (requires root)
sudo nmap -O 192.168.56.101

# Aggressive scan — OS detection, version scan, script scan, traceroute
sudo nmap -A 192.168.56.101

# Run default NSE scripts
sudo nmap -sC 192.168.56.101

# Scan all 65535 ports (slow but thorough)
sudo nmap -p- 192.168.56.101

# Scan specific port range
sudo nmap -p 1-1000 192.168.56.101

# Scan specific ports
sudo nmap -p 22,80,443,8080 192.168.56.101

# Save output to file (all formats)
sudo nmap -sC -sV -oA scan_results 192.168.56.101

# UDP scan (much slower)
sudo nmap -sU --top-ports 20 192.168.56.101
```

---

### Mistakes Made

1. **Started with `-A` scan first** — it's slow and loud. For a first pass, do a quick top-port scan then follow up with targeted scans.
2. **Forgot `-oA` flag** — didn't save the first scan. Lost the output. Now I always save output. Use `-oN` (normal), `-oX` (XML), or `-oA` (all formats).
3. **Ran nmap without `sudo` for SYN scan** — fell back to connect scan silently. SYN scan requires root because it needs to craft raw packets.

---

### What I Learned

- **SYN scan (`-sS`)** is the default when run as root — faster and harder to detect than a full TCP connect scan because it never completes the 3-way handshake.
- **Connect scan (`-sT`)** completes the full handshake — works without root but is noisier and more likely to be logged.
- **`-sC`** runs default NSE (Nmap Scripting Engine) scripts — incredibly useful for grabbing banners, checking for known vulnerabilities, and enumerating services.
- **`-oA`** saves in all three formats simultaneously — always use it.
- Version numbers in scan output are gold — they tell you exactly what CVEs to look for.
